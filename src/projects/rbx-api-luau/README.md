# rbx-api-luau
An attempt to recreate the Roblox API in pure Luau.

I'll be following the order of each class defined in the official [Roblox API](https://robloxapi.github.io/ref/).

Some parts of the repository will have functions that normally doesn't exist normally in Roblox, and I'm pretending to port it into Roblox Studio, but for now, I'll stick with this.
